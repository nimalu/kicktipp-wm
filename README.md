# kicktipp-wm

Small collection of Python tools for Kicktipp World Cup predictions and bet computation.

Quick start

- Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

- Install the project (editable):

```bash
pip install -e .
```

- Run the main script:

```bash
python compute_bets.py    # compute suggested bets
```
